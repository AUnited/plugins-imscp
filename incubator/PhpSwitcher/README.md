### REQUIREMENTS

Plugin compatible with i-MSCP >= 1.1.1

### INSTALLATION

	- Login into the panel as admin and go to the plugin management interface
	- Upload the PhpSwitcher plugin archive
	- Activate the plugin

### UPDATE

	- Login into the panel as admin and go to the plugin management interface
	- Upload the PhpSwitcher plugin archive
	- Update the plugin list through the plugin interface
