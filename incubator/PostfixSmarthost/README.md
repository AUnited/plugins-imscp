##Â PostfixSmarthost v0.0.1 plugin for i-MSCP

Plugin allowing to configure i-MSCP MTA (Postfix) as smarthost with
SASL authentication.

If you install this plugin manually, make sure it is installed in
gui/plugins/ - if the folder is called different it will not work.
